# simple-audio-codec-unity

Provides simple decoding and encoding of audio codecs by [NAudio](https://github.com/naudio/NAudio) for Unity.

## Avairable codecs

- WAVE
  - [x] Decoding
  - [ ] Encoding
- MP3
  - [ ] Decoding
  - [ ] Encoding
